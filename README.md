## parse-excel-cli

> A simple Excel parsing CLI

### Feature
```
Excel Form Data

```

- parse Excel to JSON Data

```javascript
{
    "i18n": {
        "zhCHT": {
        "KEY_1": "简体中文",
        },
        "zhCHS": {
        "KEY_1": "英文",
        },
        "en": {
        "KEY_1": "繁体中文",
        }
    }
}
```

- parse Excel to Array Data
```
[
    [
    "英文",
    "简体中文",
    "繁体中文"
    ]
]
```

### Installation

Using npm:

```
$ npm install -g parse-excel-cli
```

### Usage

```
xcli run
```
