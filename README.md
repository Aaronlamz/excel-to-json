<h1 align="center">Parse Excel CLI</h1>
A easy excel parsing CLI

## Features
### Excel to JSON

```javascript
{
    "i18n": {
        "zhCHT": {
          "KEY_1": "简体中文",
        },
        "zhCHS": {
          "KEY_1": "英文",
        },
        "en": {
          "KEY_1": "繁体中文",
        }
    }
}
```

### Excel to Array
```
[
    [
    "英文",
    "简体中文",
    "繁体中文"
    ]
]
```

## Installation

Using npm:

```
$ npm install -g parse-excel-cli
```

Using yarn:

```
$ yarn global add parse-excel-cli
```

## Usage

```
xcli run
```
